# Faydo - پلتفرم وفاداری مشتریان

پلتفرم جامع مدیریت وفاداری مشتریان با React، Tailwind CSS و Django REST API

## ویژگی‌ها

### پنل مشتریان
- **سیستم امتیازدهی:** کسب امتیاز با استفاده از تخفیف‌ها و ثبت نظرات
- **سطح‌بندی کاربران:** برنزی، نقره‌ای، طلایی، VIP با مزایای متفاوت
- **جستجوی پیشرفته:** فیلتر بر اساس دسته‌بندی، درصد تخفیف، موقعیت، امتیاز
- **پیشنهادات شخصی‌سازی شده:** بر اساس علاقه‌مندی‌ها و رفتار کاربر
- **مدال‌ها و دستاوردها:** سیستم Gamification برای تشویق کاربران
- **پروفایل شخصی:** مدیریت اطلاعات، علاقه‌مندی‌ها، تاریخچه فعالیت

### پنل کسب‌وکارها
- **داشبورد آمار:** نمایش کل درآمد، تعداد مشتری، نرخ استفاده، رضایت
- **مدیریت تخفیف‌ها:** ایجاد، ویرایش و مدیریت تخفیف‌های مختلف
- **تحلیلات پیشرفته:** آمار فروش، تقسیم‌بندی مشتریان، عملکرد تخفیف‌ها
- **مدیریت مشتریان:** نمایش فعالیت و سطح مشتریان
- **گزارش‌گیری:** تولید گزارش‌های تفصیلی از عملکرد

### امکانات کلی
- **طراحی واکنش‌گرا:** Mobile-first با پشتیبانی کامل از دسکتاپ
- **رابط کاربری مدرن:** استفاده از Tailwind CSS با طراحی مینیمال
- **احراز هویت:** سیستم ورود جداگانه برای مشتریان و کسب‌وکارها
- **چت زنده:** پشتیبانی آنلاین 24/7
- **نوتیفیکیشن:** اطلاع‌رسانی فعالیت‌ها و تخفیف‌های جدید

## تکنولوژی‌ها

### Frontend
- **React 18.3.1** - کتابخانه UI
- **Vite 5.4.0** - ابزار build و development
- **Tailwind CSS 3.4.10** - فریمورک CSS
- **React Router 7.8.1** - مدیریت مسیرها
- **Axios 1.7.9** - HTTP client برای API calls

### Backend
- **Django 5.2.5** - فریمورک اصلی
- **Django REST Framework** - API development
- **django-cors-headers** - مدیریت CORS
- **Pillow** - پردازش تصاویر
- **SQLite** - پایگاه داده (development)

## نصب و راه‌اندازی

### پیش‌نیازها
- Python 3.12+
- Node.js 18+
- npm یا yarn

### نصب Backend
```bash
# ایجاد و فعال‌سازی محیط مجازی
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# یا
.venv\Scripts\activate  # Windows

# نصب dependencies
pip install django djangorestframework django-cors-headers pillow

# اجرای migrations
python manage.py migrate

# بارگذاری داده‌های نمونه
python load_sample_data.py

# اجرای سرور
python manage.py runserver
```

### نصب Frontend
```bash
cd frontend

# نصب dependencies
npm install

# اجرای سرور development
npm run dev
```

## API Endpoints

### Authentication
- `POST /api/auth/login/` - ورود کاربر
- `POST /api/auth/register/` - ثبت‌نام

### User Management
- `GET /api/user/profile/` - پروفایل کاربر
- `PUT /api/user/profile/` - بروزرسانی پروفایل
- `GET /api/user/dashboard/` - داده‌های داشبورد

### Business Management
- `GET /api/businesses/` - لیست کسب‌وکارها
- `POST /api/businesses/` - ایجاد کسب‌وکار جدید
- `GET /api/businesses/{id}/` - جزئیات کسب‌وکار

### Offers Management
- `GET /api/offers/` - لیست تخفیف‌ها
- `POST /api/offers/` - ایجاد تخفیف جدید
- `PUT /api/offers/{id}/` - بروزرسانی تخفیف
- `POST /api/offers/{id}/use/` - استفاده از تخفیف

### Reviews & Ratings
- `GET /api/reviews/` - لیست نظرات
- `POST /api/reviews/` - ثبت نظر جدید
- `GET /api/reviews/{id}/` - جزئیات نظر

## تست کردن

### مسیر تست مشتری:
1. مراجعه به `http://localhost:5173`
2. کلیک روی "ورود" 
3. انتخاب "مشتری" و کلیک روی "👤 مشتری: testcustomer / test123"
4. کلیک "ورود" - هدایت به داشبورد مشتری
5. استفاده از منوی "جستجو" برای یافتن تخفیف‌ها
6. بررسی "پروفایل" و تنظیمات شخصی

### مسیر تست کسب‌وکار:
1. ورود مجدد با انتخاب "کسب‌وکار"
2. کلیک روی "🏢 کسب‌وکار: testbusiness / test123"
3. بررسی داشبورد کسب‌وکار با آمار کلی
4. استفاده از "مدیریت تخفیف‌ها" برای ایجاد تخفیف جدید
5. مشاهده "تحلیلات" و گزارش‌های عملکرد
